# CheXWeb



## 実行手順

```bash
# ライブラリーをインストールする
npm install

# ローカルサーバー起動 at localhost:8080
npm run serve
npm run dev

# サンプルサーバー起動
npm run serve:sample

# deploy用のビルド
# build for z環境
npm run build:dev

# build for zx環境
npm run build:sta

# build for v環境
npm run build:pro

```
